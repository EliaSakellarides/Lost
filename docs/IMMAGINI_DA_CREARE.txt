# IMMAGINI DA CREARE PER LOST THESIS
# Guida per la generazione delle immagini AI

## Stile Generale
Tutte le immagini devono essere in stile pixel art, con:
- Palette colori limitata (16-32 colori)
- Atmosfera misteriosa/survival
- Riferimenti alla serie TV LOST
- Dimensioni consigliate: 800x600 o 1024x768

## NOTE TECNICHE

1. Salva tutte le immagini in: assets/images/
2. Formato: PNG con trasparenza se necessario
3. Naming: usa esattamente i nomi file indicati sopra
4. Dopo aver creato le immagini, ricompila il gioco con: ./scripts/compile.sh
5. Testa con: ./scripts/run.sh
